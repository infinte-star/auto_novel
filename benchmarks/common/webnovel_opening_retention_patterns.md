# 网文开篇与爽点节奏结构样本

source:
- ZOL 问答《网文前三章如何写得更生动》, https://wap.zol.com.cn/ask/x_33502443.html
- ZOL 问答《网文开头如何写才合格？》, https://wap.zol.com.cn/ask/x_34132701.html
- 文小白《网文爽点设计必看！4 大黄金法则 3 经典模板》, https://blob.wenxiaobai.com/article/182b17f0-25c5-4818-2d8b-030a70e47da8
- 中国作家网相关网络文学评论 PDF, https://wyb.chinawriter.com.cn/attachment/202308/25/ddda63c5-3501-4d67-9ead-2c6251726c38.pdf

summary:
本样本只提炼公开写作建议中的结构规律，不含任何作品正文。适用于起点/番茄/七猫等连载型中文网文。

opening:
黄金三章的任务不是解释世界，而是快速交付“主角是谁、卖点是什么、冲突在哪里、为什么要追”。主角应尽早登场，差异化能力或核心设定尽早亮相，背景设定通过行动和冲突自然渗透。

chapter_1:
第1章应在较短篇幅内出现核心异常或核心危机。环境描写服务冲突，不独立铺陈。角色数量要控制，读者先记住主角、主要对手/压力源、一个关键配角即可。

chapter_2:
第2章不能只是解释第1章，应把第1章钩子转成第一个小兑现：证明主角方法有效、推翻一个错误判断、拿到一个更硬物证、或让危机升级。

chapter_3:
第3章应完成一次阶段性高潮或强反转，让读者确认本书的承诺类型。之后再开新钩子，不能连续三章只埋不兑。

payoff_pattern:
每章至少有一个可感知收益：信息收益、情绪收益、资源收益、身份收益、智斗收益。爽点需要“压迫 -> 主角行动 -> 代价 -> 兑现”，不能凭空胜利。免费阅读平台更偏高频强情绪，起点男频可以保留长线谜团，但仍要有章节推进。

notes:
禁忌：开篇两千字都在写设定；主角迟迟不登场；金手指只在简介里有，正文迟迟不用；章末并列太多问题；每章都重复同一种对峙。对悬疑题材，最有效的网文化爽点是“读者看得懂的物证反转”，不是纯解释性推理长段。
