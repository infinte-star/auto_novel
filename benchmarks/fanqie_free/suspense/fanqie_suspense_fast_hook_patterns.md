# 番茄悬疑强留存结构样本

source:
- 番茄/免费阅读平台读者偏好公开资料与网文开篇写作建议整理，不含作品正文。
- 北京大学现代出版研究所与番茄小说调研报告条目：《番茄小说免费模式读者偏好调研报告》, https://www.isss.pku.edu.cn/cbw/yjcg_1352417/yjcg_1352418/1354478.htm
- Reedsy mystery writing resources, https://reedsy.com/studio/resources/how-to-write-a-mystery
- Scholastic "Ingredients for a Mystery Checklist", https://www.scholastic.com/content/dam/teachers/lesson-plans/migrated-featured-files/jan_ingredientsmystery.pdf

summary:
番茄悬疑不能只追求本格严密，还要把本格线索转成低门槛追读刺激。每章至少出现一个“读者能看懂的反常”：尸体状态不对、物件有无不对、证词前后不对、照片/镜像/监控与现实不对、嫌疑人行为不对。

opening:
开篇优先级：异常现场 > 人物群像 > 设定解释。可以先写尸体、断联、报警无效、监控异样、死者最后一句话、主角能力第一次触发。能力规则可以边用边解释，不要一次性列硬设定。

chapter_1:
推荐结构：“强异常开场 -> 主角被迫介入 -> 封闭/时间压力钉死 -> 第一具尸体或第一反常物证 -> 能力/专业方法小试 -> 章尾抛出一个直观矛盾”。直观矛盾例：死者最后画面里有某物，但现场没有；监控显示无人进门，但门内多出湿脚印。

chapter_2:
快速兑现第一章矛盾，不要继续扩大嫌疑池。让主角用一个小物证推翻表面结论，给读者“主角有用”的确认。章末再升级：新尸体、关键人失踪、证据被毁、凶手知道主角能力。

chapter_3:
形成第一次阶段性高潮：锁定一个强嫌疑人，又马上证明“他不是唯一解释”。番茄读者需要更强的情绪起伏，本格推理也要写出人群恐慌、权威压迫、主角被质疑、主角反击。

payoff_pattern:
番茄悬疑的爽点不是长篇解释，而是“反常 -> 被误解 -> 主角指出更硬细节 -> 众人反应 -> 新危险”。每个推理点最好能被一句话说清：画面里有表，尸体没表；鞋印是湿的，嫌疑人鞋底是干的；镜中字体是正的，说明那不是镜像。

visual_contradiction_templates:
- 画面A vs 现实B：临终画面、照片、监控、镜中影像与现场状态冲突。
- 证词A vs 物证B：嫌疑人说没上楼，但鞋底沾的是五楼红锈粉。
- 群体判断A vs 主角细节B：所有人以为是高屹，因为他在五楼；主角发现手表是别人死后放上去的。
- 时间A vs 物件状态B：十分钟前声称干净的杯口，现在有新鲜口红印。

notes:
禁忌：每章只写气氛不推进；凶手线索只靠最后解释；推理过程超过一页没有人物反应；连续使用“阴影角度/光源方向”这种抽象逻辑。番茄悬疑要把复杂诡计拆成短促、可视、可感、可转发的一句话矛盾。
