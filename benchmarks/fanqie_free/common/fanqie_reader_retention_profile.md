# 番茄免费阅读读者画像与留存结构样本

summary:
番茄/免费阅读读者的决策成本低、弃书速度快，章节必须更直给。相比起点男频可慢热铺体系，番茄更需要开篇强情绪、强冲突、强可懂卖点、短周期兑现。读者不怕套路，但怕“迟迟不给收益”“解释太多”“卖点藏太深”。

reader_profile:
- 阅读入口碎片化，单次阅读耐心较短。
- 更偏好低门槛理解：冲突、身份、能力、目标必须清楚。
- 强情绪优先：委屈、危机、反差、打脸、逆袭、宠爽、惊悚、反转都要尽快落到人物处境。
- 章节收益要频繁：每章最好给一个小爽点、小反转、小线索、小升级或小危机。
- 章末钩子要单一明确，不能并列抛出太多复杂问题。

opening:
前1000字内尽量完成“主角处境 + 核心冲突 + 类型卖点”三件事。不要用长篇世界观说明开场。主角的金手指、职业优势、重生信息、系统任务、特殊感知、身份反差等，应尽早产生第一次效果。

chapter_1:
第一章要有“立即可懂的压力”。例如被冤枉、被困、被退婚、被追杀、尸体出现、系统激活、身份暴露、家庭/职场压迫。结尾最好给强钩子：主角发现关键反常、获得第一张底牌、或者危机升级到不可回避。

chapter_2:
第二章必须回收第一章钩子的一部分，不能只解释设定。读者应看到主角“真的有办法”：一次小反击、一次物证反转、一次能力验证、一次关系逆转、一次资源获得。

chapter_3:
第三章要形成阶段性确认：读者知道这本书以后主要看什么。可以是第一次打脸、第一次破案小胜、第一次系统奖励、第一次身份反差兑现、第一次感情张力爆点。第三章后再开更大的坑。

payoff_pattern:
压迫不要太长。推荐节奏是“压力 -> 反应 -> 阻碍 -> 小兑现 -> 新压力”。如果是悬疑，兑现可以是一个明确线索或推翻错误嫌疑；如果是爽文，兑现可以是打脸、升级、资源入手、身份反差；如果是女频，兑现可以是关系变化、偏爱证明、情绪反转。

notes:
禁忌包括：前三章全是设定说明；主角长期被动挨打无反制；核心卖点只在简介出现；每章结尾钩子太抽象；人物对白没有情绪立场；连续多章只铺垫不兑现。番茄读者可以接受熟悉模板，但需要更快、更清楚、更有情绪收益的执行。
