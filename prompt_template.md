## B · 故事内核【钉死】
- Logline（King 式 What if）：如果【某情境】发生，【某种人】会怎样？——全书试金石，后续每个选择都要服务于它

## C · 人物驱动力【钉死骨架 · 留白具体反应】
- 主角状态机：欲望 / 恐惧 / 致命弱点 / 成长方向 / **人设记忆点**（1-3个可演的标志：口头禅、标志动作、外形细节、反差萌；禁"善良/聪明"等空泛词）
- 核心配角 ×2-3：各一行「想要什么 + 底线在哪 + 能力边界 + 立场/利益 + 记忆点」
- 主线反派 / 幕后：自洽动机 + 能力上限（**严禁降智送人头**）
- **群像阵容表【群像书填写，单主线/单 CP 书跳过】**：把每一位分量相当的成员（多男/多女追求者、团队群像等）逐个列全，**一个都不能省**，每人一档：
  · 关系定位 / 追求方式（合伙 / 守护 / 治愈 / 直球 / 灵魂知己 / 高智占有 / 青梅归属……，**各人绝不重复**）
  · 反差与缺口（人前 vs 人后 + 他自己要克服的成长缺口，非完美工具人）
  · 入场符号（一个专属道具/动作/台词习惯，读者一眼可辨，如虎口旧疤、粉色头盔、口头一个"姐"）
  · 专属名场面（种子）（只属于他、最上头的一个高光/破防瞬间雏形）
  · 一句话声音辨识（钉死说话质感：短而稳 / 贫而正 / 柔而清 / 冷而锐 / 温而狠 / 熟而暖……全员开口即可辨）
  去同质化铁律：多人**不得做同一种好、同一件事、说同一类话**；价值由职业/性格/共同历史唯一决定。
- 矛盾矩阵 ≥3组：列出相互拉扯、能交叉施压的矛盾对（个人意志vs系统惯性 / 短期爽vs长期代价 / 情义vs利益 …）

## D · 爽点与情绪系统【留白 · 只定方向不写死桥段】
- 爽点主轴：偏哪类（打脸 / 暴富 / 升级 / 解谜 / 制度 / 身份反差 / 情感 …）
- 爽点轮换表：每章命中一类、如何循环（例：破陷阱→借刀反杀→打脸→藏锋→卷末抛更大局）
- 压抑↔兑现比例：按 §0 档位（快=当章前半立冲突、后半即兑现；慢=更长铺垫、兑现更稀更重）
- 情绪曲线：每卷情绪起伏走向（压抑积累→高潮爆发→余味钩子）；每次兑现都留更高层级新问题
- **关系线高光错峰【群像书填写】**：多成员的高光如何错峰分配——每章聚焦谁、谁用钩子后补，核心成员不得连续两章隐形，不追求每人每章等量分镜（宁可集中给足，不要平均摊薄）
- **争宠 / 多线升级曲线【群像书填写】**：关系竞争的强度必须**逐章升级**（如隐性较劲→关系特权/专业能力碰撞→正面同框），严禁三章同一水位；每次同框都要产出新信息，禁止只写"大家吃醋"而无后续
- **心动破防节拍【情感向书填写】**：每章**恰好 1 处**"读者独享"的心动破防——用生理反应+嘴硬掩饰呈现（耳朵红/眼眶热/攥紧某物却说别的），让读者比角色先懂；逐章破防方式要有变化，不得同质复读

## E · 结构 · 节奏 · 伏笔【钉死骨架】
- 黄金三章：逐章各一句话，写出**每章独立的钩子 / 当章兑现**（前3章决定弃书率，引擎对此段做更严审查）
- 卷纲骨架（前2卷详写、其余各1段概要）：每卷含 主题 / 阶段高潮（每15-25章一个）/ 大事件锚点（≥2-3个具体事件，点名涉及哪些人物）/ 可见代价 / 遗留危机（= 下一卷核心矛盾来源）
- **伏笔错峰兑现表**（核心长线严格控制在 3-5 条）：逐条标注「埋在哪 → 早期小揭示 → 中期推进 → 最终兑现」的**章节节点**；兑现节点必须**错峰分散（间隔≥2-3章）**，**每章只收束1条主线索，严禁多条兑现挤进同一章或收束点相邻几章**（挤兑=该章过载=模型写不动=崩章）
- **角色高光轮值 & 反转做成名场面排期【群像书填写】**：逐章标注①该章哪些成员领独立高光及其类型（去同质化）；②每个身份/关系/命运反转的**引爆章号**——错峰分散、一章最多引爆一个重量级反转，且每个都必须落地为有对话/动作/破防反应的**当章场景**，禁止写成旁白背景说明

## F · 开写铁律【★ 最高优先级 · ≤3条 · 每章开写前逐条自检 · 违一即废稿】
从「黄金三章 / 招牌设定 / 最易违约点」里挑最要命的 ≤3 条，每条短、具体、可逐章判定（例：①本副本生存规则必须在早段用编号清单逐条明示；②主角零战力，破局只准靠推理/常识/借力，禁止亲自打斗驱邪；③金手指只被动给残缺提示、每用一次付代价，禁直接给通关答案）。没有就留空。

## G · 硬性禁止项【红线护栏】
明令禁止的套路（第一定律的反面）：不许主角全知全能 / 凭空获得没铺垫的力量破局；不许反派降智；不许已立约束凭空消失；不许重大胜利零代价；不许靠巧合或天降救兵解决主线危机。按题材补充更多。
