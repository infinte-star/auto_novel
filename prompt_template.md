# 小说设定（混合派创作纲要）
#
# 这是整个写作流水线唯一的"创作纲要"输入，bootstrap 阶段据此生成
# 世界观/人物/时间线/伏笔/卷纲 等记忆文件。
#
# ── 混合派写作理念（Hybrid：架构派 × 探索派）──────────────────
# 顶级作家不是纯架构派(Sanderson/Rowling)就是纯探索派(King/Martin)，
# 而是混合：把"必须钉死的"钉死，把"该留白的"留白。本模板据此分两区：
#
#   【硬骨架 · 必须钉死】世界规则、能力边界、矛盾矩阵、卷纲锚点、兑现台账、硬约束。
#       —— 决定"开挂会不会崩坏""伏笔会不会烂尾"，越具体越好。
#   【探索留白 · 给引擎自由】人物在情境中如何反应、对白机锋、场景细节。
#       —— 只给"情境(What if)"和"人物动机"，不写死过程，留给逐章生成发挥。
#
# 黄金法则：
#   1. 写"限制"重于写"能力"——主角不能做什么，比能做什么更出戏(Sanderson 第二定律)。
#   2. 能力解题的合理度，正比于规则被讲清的程度——别用没铺垫的力量强行破局(第一定律)。
#   3. 每次兑现都留下更高层级的新问题；每次重大胜利都付出可见代价。
#
# 删掉本行及上面的注释，认真填写后即可运行。

## 类型
（例：玄幻 / 都市 / 历史改写 / 科幻 / 悬疑 …，可叠加标签）

## 一句话前提（Logline）
（King 的 "What if…"：如果【某情境】发生，【某种人】会怎样？用一句话说清。
这是全书的试金石——后续每个选择都要能回答"它服务于这个前提吗"）

## 核心命题
（这本书到底在讲什么、主角面对的根本困境是什么、爽点来源是什么）

## 世界规则与硬设定（Hard Rules）
（这是"硬"的部分，必须像科学定律一样清晰、可被读者预期：
- 力量/能力体系的运作规则与代价（用一份资源/精力/风险换取效果）
- 社会结构、权力秩序、阵营关系
- 资源稀缺、时间压力等硬约束
规则越清晰，主角靠"巧用已知规则"破局时读者越觉得"赚到了"而非"开挂"）

## 主角的能力边界与代价（重点写"不能"）
（姓名、身份、性格底色、成长方向。强调三点：
- 初始能力边界——主角绝不能一开始就全知全能；
- **致命弱点 / 限制**——他做不到什么、用能力要付什么代价（比能力更出戏）；
- 成长曲线——能力逐卷扩张，但每一步都伴随新约束，不是无脑变强）

## 主要矛盾（矛盾矩阵）
（列出多组相互拉扯的矛盾，如 个人意志 vs 系统惯性、短期爽感 vs 长期代价、
情义 vs 利益 …。至少 3 组，且彼此能交叉施压）

## 群像（人物状态机）
（每个重要人物给一行："想要什么 + 底线在哪 + 能力边界 + 立场/利益"。
重要反派必须有自洽动机与能力上限——严禁反派降智来送人头）

## 长篇结构（节拍）
- 每 3-5 章一个小兑现。
- 每 15-25 章一个阶段高潮。
- 每 60-80 章一卷，每卷解决一个主要矛盾并开启更高层级危机。
- 每卷至少 2-3 个大事件锚点（具体事件，不是抽象目标）。
- 重大胜利必须有代价，重大失败必须留后续修复路径。

## 卷纲方向与各卷代价
（列出前几卷各自的：主题 / 阶段高潮 / **本卷的可见代价**（资源/关系/信任/身份的损失）/
卷末开启的更高层级危机。把"代价"落到具体事件，不要只写口号）

## 伏笔与兑现台账（Setups & Payoffs）
（仿 Rowling 章节网格的思路，列出关键伏笔：埋在哪、大致何时兑现、兑现后引出什么新问题。
引擎会自动追踪沉默伏笔与兑现间隔，这里给出主干即可）

## 硬性约束（明令禁止）
（明确禁止的套路，这是第一定律的反面护栏：
- 不许主角全知全能 / 不许凭空获得没铺垫的能力解题；
- 不许反派降智；不许已立的约束凭空消失；
- 不许重大胜利零代价；不许靠巧合/天降救兵解决主线危机。
按你的题材补充更多禁止项）

## 爽点设计（情绪主轴，不写死桥段）
（这本书的爽点偏向哪类：战斗爽 / 战略爽 / 打脸爽 / 制度爽 / 情感爽 …。
只定"读者要的情绪"，具体怎么兑现交给逐章生成探索）

## 压抑与兑现比例
（如"七分压迫，三分兑现"。每次兑现都要留下新的更高层级问题）

## 探索空间（鼓励引擎发挥的留白）
（明确告诉引擎：哪些地方欢迎即兴发挥、人物可以"自己长出来"。例如：
- 配角的具体反应与对白机锋——只要符合其动机与底线，鼓励意外但合理的选择；
- 单场景的画面、细节、节奏——不必预设；
- 支线的具体走向——只要不违反硬骨架与硬约束。
这一段是把 King 式"把人物丢进情境看他怎么逃出来"的自由度显式授权给引擎）

## 文风要求（声音锚）
（叙事基调、对话风格、节奏偏好；可读性与厚重感如何平衡。
可给一两句"理想语感"的样例，作为全书声音锚）
