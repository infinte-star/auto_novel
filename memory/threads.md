1. **【辽东军饷】**
   - **引入**：登基初，户部奏报辽东欠饷已达半年，边军怨言四起。
   - **预期**：皇帝必须设法筹集一笔银子稳住辽东防线，同时探查欠饷的真正原因（是真没钱，还是被贪墨挪用）。
   - **状态**：进行中，已通过暗账初步发现漕运、盐税环节存在巨大漏洞。

2. **【漕运命脉】**
   - **引入**：秋粮北运关键时刻，漕运总督奏报河道淤塞、沿途损耗巨大。
   - **预期**：漕运问题直接关系京城粮食与财政安全，牵涉沿线无数官吏、勋贵、士绅利益。整顿漕运是后续一系列改革的关键支点，也是第一个触及核心利益集团的行动。
   - **状态**：伏笔已埋，皇帝通过密折开始了解详情。

3. **【锦衣卫残影】**
   - **引入**：李若琏奉命秘密调查某件与魏忠贤旧案相关、但疑点重重的小案。
   - **预期**：此案将牵扯出魏忠贤时期被掩盖的另一桩涉及边将或财政的秘密，成为皇帝了解帝国另一面的窗口，并可能借此案重建一个精干的直属情报小队。
   - **状态**：调查已启动，初步线索指向一个中层官员。

4. **【东林真容】**
   - **引入**：在“清查阉党”议题上，东林党人表现积极，但其内部对“清查范围”和“后续政策”存在分歧，且部分骨干与江南巨富关系密切。
   - **预期**：皇帝将逐渐发现东林并非铁板一块的“清流”，其背后也有地域、商业利益。这将成为他未来分化利用文官集团的关键认知。
   - **状态**：朝堂辩论已初现端倪。

5. **【将门利益】**
   - **引入**：辽东方面要求增加“火器”与“棉甲”拨付，但兵仗局报告产能不足、质量堪忧。
   - **预期**：军事装备从生产到运输到使用，形成一条庞大的利益链。皇帝试图整顿军工，将触及工部、兵仗局、内廷、运输官员乃至边军采买系统的利益。
   - **状态**：问题已上报，等待皇帝决策。

6. **【西北火星】**
   - **引入**：陕西某县令密奏，地方大旱，流民聚集，请求赈济，但奏折被户部以“灾情不实”驳回。
   - **预期**：这是西北大乱的最初火苗。皇帝如何回应（是压下来，还是设法了解真相并尝试微小干预），将决定这颗火星是暂时被掩盖，还是被谨慎地监控起来。
   - **状态**：密折在御案，皇帝面临选择。
